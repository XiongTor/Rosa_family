# 如何判断当前系统发育树的ILS严重程度，以及是否能通过单纯的增加数据的ILS程度恢复原本的系统发育树

# 具体实现步骤如下：
# 1. 通过IQTREE对ASTRAL得到的物种树进行枝长重新估算，得到带有实际枝长的物种树
# 2. 通过IQTREE与ASTRAL树的枝长比较，得到theta值的分布范围
# 3. 通过SimPhy模拟不同theta值下的基因树
# 4. 计算实际基因树与模拟基因树的所有四分体组合，并进行比较，判断每一个四分体其实际出现频率与模拟出现的频率。并进行相关性回归分析
# 5. 汇总最终的回归系数，判断哪个theta值能够最好的拟合当前的实际数据结果，并通过其拟合程度，判断当前系统发育树的ILS严重程度，以及是否能通过单纯的增加数据的ILS程度恢复原本的系统发育树
